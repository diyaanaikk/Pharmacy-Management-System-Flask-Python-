# Pharmacy-Management-System-Flask-Python-
A web-based Pharmacy Management System built using Python and Flask, designed to manage medicine inventory efficiently with real-time search, smart filtering, and a user-friendly dashboard.
Features
Add, update, and delete medicines
Real-time live search (no page reload required)
Filter by supplier, stock level, and expiry date
Low-stock and expired medicine alerts
Dashboard with inventory statistics
Responsive UI using Bootstrap
Database management using SQLAlchemy
SQLite / MySQL support
